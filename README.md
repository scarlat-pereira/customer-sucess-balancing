# Customer Success Balancing

## Descrição

Este projeto oferece uma solução para o problema de balanceamento de Customer Success.

## Instruções para Execução

1.  Abra o terminal e crie um diretório no local de sua preferência com o comando **mkdir**:

```javascript
mkdir scarlat-solution-challenge
```

2. Entre no diretório que acabou de criar e depois clone o projeto:

```javascript
  cd scarlat-solution-challenge
  git clone git@github.com:scarlat-pereira/customer-sucess-balancing.git
```

3. Acesse o diretório do projeto e depois utilize o comando **yarn** ou **npm i** para instalar todas as dependências necessárias:

```javascript
  cd customer-success-balancing
  yarn
  npm i
```

4. Para rodar os testes, digite um dos comandos abaixo:

```
yarn test
npm test
```
